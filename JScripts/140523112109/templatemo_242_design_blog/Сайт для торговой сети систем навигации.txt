Сайт для торговой сети систем навигации

Цель сайта:
1. Продать навигационные системы
2. Сообщать о местонахождении магазинов

Планируемые страницы сайта:
1. Главная
2. Каталог
3. Страница администрирования
4. Корзина
5. личный кабинет
6. Контакты

общие элементы всех станиц:
1. Шапка с сылками на оставшиеся страницы
2. Сайд бар с информацией покупателя ( на обсуждении как он будет выглядить)
3. Область контента

Задача: 
Составить план содержания страницы:
    1. Новости, горячие предложения, реклама, ссылки на партнеров магазина
    2. карточка: название, описание, картинки, цена, скидки, количество
    3. Запросы редакирования, добавления, удаления товаров, новостей, пользователей
    4. Инфо покупателя, товары в корзине, цена, количество, скидка
    5. Инфо покупателя, регистрационные данные, история покупок, промокоды и скидки
    6. Контактная информация о мазазинах, партнерах, лицензии и т.п
Составить требования к API для работы страниц:
    1. Авторизация:
        1.1 Покупатель:
            1.1.1 Просмотр страниц кроме администрирования
            1.1.2 Добовление товаров в корзину
            1.1.3 Редактирование корзины (количество, ассортимент,)
            1.1.4 Редактирование информации о себе, способы оплаты, доставка
            1.1.5 Оплата заказа

        1.2 Сотрудник(администратор):
            1.2.1 Просмотр страниц 
            1.2.2 Редактирование корзин пользователей
            1.2.3 Редактирование карточек (количество, ассортимент,)
            1.2.4 Редактирование информации о себе, пользователей, способы оплаты, доставка
            1.2.5 Редактирование контента главной страницы
